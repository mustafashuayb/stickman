# Stickman game

## Description
- This is my stickman game i created in Visual Studio Code. 
- The game is about a stickman running left and right and having to jump on to platforms.  

## How To Run
- Make sure Python 3.14 is installed. If it isn't, open your respective link:
    - [Python for Windows](https://www.python.org/downloads/windows/)
    - [Python for Mac](https://www.python.org/downloads/windows/macos)
    - [Python for Linux](https://www.python.org/downloads/source/)
    - [Python for Another Other Device](https://www.python.org/downloads/other/)

- Open your python editor and run the following in your python terminal:
    - python stickmangame.py

## Controls 
- Press the left and right arrow keys on your keyboard to move the figure left and right respectively.
- Press the space bar on your keyboard to move the figure up.
